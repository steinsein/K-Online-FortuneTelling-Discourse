# 05_visualization — 시각화

분석 결과를 논문용 그래프로 시각화하는 스크립트와 생성된 이미지 파일입니다.

## 스크립트 (`scripts/`)

| 파일 | 설명 |
|------|------|
| `platform_comparison.py` | 플랫폼별 비교 차트 (도넛, 버블, 히트맵 등) |
| `temporal_trends.py` | 시계열 추이 그래프 (분기별 게시물 수) |
| `sentiment_transform_plot.py` | 전체 데이터 감정 분석 결과 시각화 |
| `sentiment_transform_plot_50k.py` | 후기 데이터(~50,000건) 감정 분석 결과 시각화 |

> `04_analysis/`의 `platform_comparison.py`, `temporal_trends.py`와 시각화 코드가 일부 통합되어 있습니다. 분석과 시각화가 하나의 스크립트에서 수행되는 경우도 있습니다.

## 시각화 설정

- 한글 폰트: Malgun Gothic (Windows) / NanumGothic (Linux/Mac)
- 컬러 팔레트: 플랫폼별 고유 색상 지정
  - 네이버블로그: `#2DB400`, 네이버카페: `#00C73C`
  - 디시인사이드: `#3b5a99`, 에펨코리아: `#4682B4`
  - 더쿠: `#D81B60`
- 출력: PNG (300 DPI)
